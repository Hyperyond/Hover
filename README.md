# Hover

> AI that hovers over your dev workflow. 
> Natural language frontend testing, embedded in your dev environment.

⚠️ This project is in early development. Stay tuned.

## What is Hover?

Hover is a Vite plugin that lets you test your frontend by talking to it. 
While developing, open the floating chat in your browser, describe what 
you want to verify in plain English, and watch AI operate your app directly.

Every verified scenario can be crystallized into a reusable Playwright test 
— so you build a test suite without ever writing test code.
