# Running examples

Each example app under `examples/*` is a testing-surface target the extension (or `pnpm smoke`) drives over CDP. Run any of them after `pnpm install`:

```bash
pnpm dev:example:basic-app           # http://localhost:5173 — login + counter + todos
pnpm dev:example:e-commerce          # http://localhost:5174 — Amazon-style storefront
pnpm dev:example:stock-registration  # http://localhost:5175 — brokerage account form
pnpm dev:example:canvas-paint        # http://localhost:5176 — canvas + DOM toolbar
pnpm dev:example:payment-provider    # http://localhost:5177 — third-party popup target
```

The bundler-dogfood examples (`astro-app` / `nuxt-app` / `next-app` / `webpack-app` / `rn-web-app`) and `turbo-monorepo` were removed along with the bundler-plugin path.

The examples install nothing Hover-related — they're plain Vite + React dev servers; the VS Code extension drives any of them over CDP. Launch the isolated debug Chrome with `pnpm smoke:chrome` (port `:9222`, profile under `<tmpdir>/hover-chrome`, which reuses session state across runs).

## Auxiliary commands

```bash
pnpm smoke:chrome     # launch debug Chrome without an example dev server
pnpm smoke            # end-to-end: detect agents → CDP preflight → invoke claude
pnpm detect           # list installed coding-agent CLIs
pnpm ws-smoke         # exercise the @hover-dev/core WebSocket bridge in isolation
pnpm bench-ttfb       # time the LLM-driven loop's first tool_use latency
```

## Multiple examples at once

The examples are independent dev servers, so running several at once is fine — each binds its own Vite port. The extension's engine binds `127.0.0.1:51789` and auto-bumps (`:51790`, …, up to `:51798`) if that port is busy.

Running e-commerce + payment-provider together is supported; the e-commerce example's *Pay with PayHover* popup opens the payment-provider tab on `:5177`. Payment-provider simulates a third-party origin.

## Common environment overrides

```bash
HOVER_AGENT=codex pnpm dev:example:basic-app  # default to codex instead of claude
HOVER_MODEL=opus  pnpm smoke                  # one-off opus run (default is sonnet)
HOVER_CDP=http://localhost:9333 pnpm smoke    # different debug Chrome
```
